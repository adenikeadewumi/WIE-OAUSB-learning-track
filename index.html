Four-Saturday Curriculum

Participants commit to one track. Each session builds on the last.

A

AI & Software Engineering

From zero code to a working intelligent system

Beginner – Intermediate Laptop Required Google Colab
Saturday 1

Python Fundamentals & Data Thinking

  • Python setup via Google Colab — no installs needed
  • Variables, loops, functions — practical syntax
  • Intro to pandas: loading & exploring a real dataset
  • Data visualisation with matplotlib
  • Mini challenge: clean a messy Nigerian market dataset
✦ Leave able to write and run Python scripts independently
Saturday 2

Machine Learning Without the Math

  • What is ML? Supervised vs unsupervised — plain language
  • Train an image classifier with Teachable Machine
  • Intro to scikit-learn: a simple classification pipeline
  • African use cases: crop disease, fraud, health triage
  • Model evaluation: accuracy and confusion matrix
✦ Train and evaluate your first ML model
Saturday 3

NLP & Prompt Engineering

  • How large language models work — conceptual overview
  • Prompt engineering: zero-shot, few-shot, chain-of-thought
  • Calling an AI API with Python (live demo)
  • Build a simple AI-powered tool: Q&A bot or summariser
  • Ethics and bias in AI systems
✦ Build and run a simple AI-powered application
Saturday 4 — Capstone

Build & Pitch

  • Teams define a real problem — campus, community, industry
  • Prototype: an ML model or AI-powered script
  • Present to a panel — 5 minutes + Q&A
  • Prizes for top projects
  • Certificate presentation
✦ Present a working AI prototype to a live panel
B

Robotics & Embedded Systems

Hardware comes alive — build things that move and sense

All Levels Arduino + Raspberry Pi Quanser on S4
Saturday 1

Electronics Foundations & Arduino

  • Circuit basics: voltage, current, resistance — hands-on
  • Arduino IDE setup and first sketch (Blink)
  • Digital & analog I/O: LEDs, buttons, potentiometers
  • Sensors: ultrasonic distance, temperature (DHT11)
  • Challenge: build a proximity alarm
✦ Wire and program a sensor-driven Arduino circuit
Saturday 2

Microcontrollers & Actuator Control

  • Servo & DC motor control with PWM signals
  • Intro to Raspberry Pi: GPIO and Linux terminal basics
  • Serial communication between Arduino and Pi
  • Camera module: capturing frames with OpenCV
  • Mini build: camera-triggered motor response
✦ Control a motor or servo from a Raspberry Pi script
Saturday 3

Computer Vision for Robotics

  • OpenCV fundamentals: colour filtering and contours
  • Object detection with YOLOv8n on Raspberry Pi
  • Coordinate mapping: pixel space to world coordinates
  • Integrating detection output with motor control logic
  • Live demo: vision pipeline triggering an actuator
✦ Run a vision pipeline that triggers a physical response
Saturday 4 — Capstone

Quanser Demo & Final Build

  • Intro to Quanser: industry-grade control systems
  • Guided demo: balance or position control on Quanser hardware
  • Teams present their S1–S3 build project
  • Panel feedback and awards
  • Certificate presentation
✦ Interact with industry hardware and present a build
C

Engineering Design & CAD

From sketch to technical drawing — precision meets design thinking

All Disciplines AutoCAD + Fusion 360 CV-Ready Skill
Saturday 1

Engineering Drawing Principles

  • Why technical drawing matters: industry & communication
  • Orthographic projection: front, top, side views
  • Dimensioning standards and title blocks
  • Hand sketching: 2D shapes to 3D objects
  • Introduction to AutoCAD interface and navigation
✦ Produce a correctly dimensioned 2D orthographic drawing
Saturday 2

2D Drafting in AutoCAD

  • Draw, modify, and annotate tools in AutoCAD
  • Layers, line types, and drawing standards
  • Precision drawing: snap, grid, object tracking
  • Floor plan or mechanical part exercise
  • Printing to scale: layouts and viewports
✦ Produce a print-ready technical drawing in AutoCAD
Saturday 3

3D Modelling in Fusion 360

  • Fusion 360 setup — free student licence
  • Sketch → Extrude → 3D solid modelling workflow
  • Fillets, chamfers, holes, and assembly basics
  • Rendering and visual presentation
  • Export for fabrication: STL and DXF formats
✦ Model a functional 3D component ready for fabrication
Saturday 4 — Capstone

Design Challenge & Presentation

  • Teams receive a design brief — bridge, bracket, or component
  • Full pipeline: sketch → 2D drawing → 3D model
  • Present design rationale and drawings to panel
  • Judged on accuracy, clarity, and design thinking
  • Certificate presentation
✦ Deliver a complete design package from brief to 3D model
